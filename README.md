# CodeB Tool (구 DIY)

### DIY와 CodeB 병합

## Installation
1. NPM 모듈 설치
```
npm install
```

2. 하단의 커맨드로 서버 실행
```
npm run start
# or
node server.js
# when using dev
npm run dev
```
