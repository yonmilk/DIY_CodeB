# CodeB Tool (구 DIY)

### DIY와 CodeB 병합

## Installation
1. NPM 모듈 설치
```
npm install
```

2. 하단의 커맨드로 서버 실행
```
# 하단 환경은 세션 유/무에 따른 접근 제어일 뿐, 그 외에 특별한 기능은 없음.

# 개발 환경일 시
npm run dev
# 배포 환경일 시
npm run prod
```
